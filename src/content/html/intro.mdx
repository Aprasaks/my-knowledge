# HTML 소개

HTML은 **HyperText Markup Language**의 약자입니다.  
웹페이지의 구조를 만드는 데 사용됩니다.

## 주요 특징

- 마크업 언어 (Markup Language)
- 태그(tag) 기반 구조

## 기본 예제

아래는 가장 기본적인 HTML 문서 예시입니다.

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>예시 페이지</title>
  </head>
  <body>
    <h1>나의 첫 번째 HTML 페이지</h1>
    <p>Hello, world!</p>
  </body>
</html>
```
